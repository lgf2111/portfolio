## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/lgf2111/portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Running

1. Run the server
   ```sh
   npm run dev
   ```
